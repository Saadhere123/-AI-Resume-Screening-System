🤖 AI Resume Screening System
📌 Project Description

The AI Resume Screening System is an intelligent web application that automatically analyzes and ranks candidate resumes based on their relevance to a given job description. It uses Machine Learning (TF-IDF Vectorization + Cosine Similarity) to compare resumes with job requirements and generate an accurate match score.

This project helps recruiters and HR teams to quickly shortlist the most relevant candidates by automating the resume screening process, saving time and improving hiring efficiency.

Built using Python, Gradio, and Scikit-learn, the system provides a simple and interactive interface where users can upload multiple PDF resumes and instantly get AI-powered ranking results.

🚀 Features
📄 Resume Processing
Upload multiple PDF resumes

Clean and process resume data for analysis
🧠 AI-Based Matching System
TF-IDF vectorization for text representation
Cosine similarity to calculate job-resume match
Smart ranking based on relevance score
📊 Ranking & Analytics
